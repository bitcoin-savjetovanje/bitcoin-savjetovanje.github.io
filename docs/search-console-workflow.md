# Search Console workflow

Ovaj kratki workflow služi za SEO odluke nakon što Google pokupi nove stranice.

## Tjedni pregled

1. Otvoriti Google Search Console za `bitcoin-savjetovanje.com`.
2. U Performance izvještaju gledati zadnjih 28 dana.
3. Filtrirati stranice:
   - `/bitcoin-savjetovanje/`
   - `/bitcoin-konzultacija/`
   - `/razgovor/`
   - `/vodici/`
   - najjače vodiče po impressions
4. Za svaku stranicu zapisati:
   - queryje s impressions
   - poziciju 8-20
   - CTR ispod 2%
   - queryje koji ne odgovaraju namjeri stranice

## Što mijenjati

- Ako stranica ima impressions, ali slab CTR: prvo promijeniti SEO title ili meta description.
- Ako je pozicija 8-20: dodati kratak odlomak koji izravno odgovara queryju, bez punjenja ključnim riječima.
- Ako query vodi na krivu stranicu: dodati interni link prema boljoj stranici.
- Ako query traži porezni, pravni ili investicijski savjet: ne loviti ga agresivno; zadržati jasnu granicu usluge.

## Mjesečni pregled

Jednom mjesečno odabrati najviše tri stranice za doradu. Ne mijenjati sve odjednom, jer se inače ne vidi što je pomoglo.

Prioritet:

1. Stranice s komercijalnom namjerom: `bitcoin savjetovanje`, `bitcoin konzultacija`, `bitcoin savjetnik`.
2. Vodiči koji dobivaju impressions, ali nemaju klikove.
3. Stranice koje imaju klikove, ali slab nastavak prema `/razgovor/`.

## Biljeske za ovaj site

- Primarni cilj je uvodni razgovor, ne masovni promet.
- Bolje je privući manji broj ljudi s pravim pitanjem nego širiti copy prema ljudima koji traže signal, prognozu cijene ili upravljanje sredstvima.
- Fraze koje vrijedi pratiti: `bitcoin savjetovanje`, `bitcoin konzultacija`, `bitcoin savjetnik hrvatska`, `bitcoin sigurnost`, `bitcoin nasljeđivanje`, `kako čuvati bitcoin`, `bitcoin dug`.
